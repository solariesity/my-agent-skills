# Writing Style Guide (学术/作业类专有)

**范围限定：仅用于作业、报告、论文等学术或任务型写作。不适用于创意写作、营销文案、社交媒体等其他类型。**

## 触发条件

用户说出以下关键词或类似表述时，自动应用此规范：
- "帮我写一段文字/文章/报告"
- "写个作业" / "写个课程论文"
- "关于 XX 的评价/影响/分析"
- 上下文明显指向学术写作或任务型写作

当用户说"写个故事""写个文案""写个广告"等非学术场景时，不要触发此 skill。

## 中文写作规范

### 结构要求
- 尽量少用分点列举（1. 2. 3.）。如需列举，优先用连贯段落自然展开。
- 段落之间要有逻辑衔接，不用生硬的"首先/其次/最后"。
- 文章结构清晰即可，不需要刻意"包装"。

### 语言风格
- 朴实、严谨，不花哨。
- 不用网络流行语、不用过度修辞（如"犹如""仿佛""堪称"等）。
- 不用"总之""综上所述"等八股式结尾。
- 词汇和句式以中国大学生/研究生日常写作水平为准，不刻意拔高，也不过于口语化。
- 避免重复用词，但也不需要刻意在乎"文采"。

### 语气
- 客观、平实，像一篇合格的课程论文或调研报告。
- 不卑不亢，不讨好，不说废话。

## 英文写作规范

### 词汇要求
- 不使用高级/生僻词汇。优先使用 CET-4/CET-6 范围内的常用词。
- 避免 GRE/SAT 级别的生僻词。
- 一个意思如果有简单词和高级词，选简单词。

### 语法要求
- 尽量少用复杂语法结构。
- 避免过多从句嵌套。优先使用简单句和并列句。
- 避免使用倒装、省略、虚拟语气等高级语法，除非必要。
- 时态以一般现在时和一般过去时为主，不需要刻意变换。

### 整体水平
- 以中国大学生/非英语专业研究生水平为准。
- 目标是"清楚、准确、自然"，而不是"炫技"。

## 执行要点
- 收到符合触发条件的写作任务后，直接用上述规范的语言输出，不需要先说明"我将采用某某风格"。
- 如果用户明确要求改变风格（如"用更正式的语气""用英文高级词汇"），以用户最新要求为准。
- 写完后可自行检查：有没有不必要的高级词？有没有堆砌分点？如果有，改掉再输出。

## 未来扩展说明
此 skill 专用于学术/作业类写作。用户可能会创建其他 skill 分别管理不同场景（如创意写作、营销文案等），届时按相应 skill 的触发条件执行。
