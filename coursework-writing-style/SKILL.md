# Coursework Writing Style Guide（课程作业写作规范）

**范围限定：用于帮助学生完成课程作业、课程论文、实验报告、读书报告、反思等学生提交物。不适用于发表级学术论文、创意写作、营销文案、社交媒体内容。**

## 触发条件

当用户请求涉及以下场景时，调用本 skill：

- "帮我写这门课的作业/报告/课程论文"
- "这个作业要求分析 XX，帮我写一下"
- "课程论文需要 N 字，主题是 XX"
- "帮我写个实验/实践报告"
- "帮我写个读书报告/文献综述"
- "帮我写个课堂 reflection/讨论回应"
- 上下文明显指向完成课程任务

不应触发本 skill 的场景：

- "帮我写篇 SCI/顶会论文" → 使用论文写作 skill
- "写个故事/小说" → 创意写作
- "写个广告/小红书文案" → 营销/社交媒体写作
- "写条生日祝福" → 私人写作

### 触发示例

应该触发：
- "帮我写个 2000 字的课程论文，主题是人工智能对教育的影响。"
- "这个实验报告需要描述结果和讨论，数据在附件里。"
- "帮我写个 500 字的课堂反思。"
- "这门课的小组报告要求分析三个案例，帮我搭个框架。"

不应触发：
- "帮我写篇 Nature 投稿的 Introduction。" → 不是课程作业
- "帮我写个科幻短篇。" → 创意写作
- "帮我写个产品推广文案。" → 营销文案

## 写作前先对齐作业要求

开始写作前，先确认以下信息。如果用户未提供，主动询问：

- **题目/主题**：具体写什么？
- **字数要求**：总字数、各段落大致比例？
- **格式要求**：是否需要摘要、引言、结论？引用格式？
- **提交形式**：个人作业还是小组作业？是否需要分工说明？
- **评分标准**：老师看重论点、结构、还是分析深度？

如果用户已经提供了作业要求文件或截图，先读取再动笔。

## 按作业类型调整语气

| 类型 | 重点 |
|---|---|
| 课程论文 | 有论点、有结构，像小型论文，但语言保持学生水平 |
| 实验/实践报告 | 客观描述过程、结果和讨论，不过度发挥 |
| 读书报告/文献综述 | 准确概括原文，避免过度评价 |
| 反思/讨论回应 | 可以带个人思考，但仍需逻辑清晰、言之有物 |
| 小组作业 | 注意表述一致性，避免人称混乱 |

## 中文写作规范

### 结构要求
- 尽量少用机械分点（1. 2. 3.）。如需列举，用连贯段落自然展开。
- 段落之间用过渡句衔接，不用生硬的"首先/其次/最后"。
- 文章结构清晰即可，不需要刻意"包装"或"拔高"。

### 语言风格
- 朴实、严谨，不花哨。
- 不用网络流行语、不用过度修辞（如"犹如""仿佛""堪称"等）。
- 不用"总之""综上所述"等八股式结尾。
- 词汇和句式以中国大学生/研究生日常写作水平为准，不刻意炫技，也不过于口语化。
- 避免重复用词，但也不需要刻意追求"文采"。

### 语气
- 客观、平实，像一篇认真完成的课程作业。
- 不卑不亢，不讨好，不说废话。

## 英文写作规范

### 词汇要求
- 优先使用常用词，避免过度生僻的学术词汇。
- 一个意思如果有简单词和复杂词，选简单词。

### 语法要求
- 尽量少用复杂语法结构。
- 避免过多从句嵌套。优先使用简单句和并列句。
- 时态以一般现在时和一般过去时为主。

### 整体水平
- 以中国大学生/非英语专业研究生水平为准。
- 目标是"清楚、准确、自然"，而不是"看起来像母语者写的"。

## 执行要点
- 收到写作任务后，直接按上述规范输出，不需要先说明"我将采用某某风格"。
- 如果用户明确要求改变风格（如"用更正式的语气""用英文高级词汇"），以用户最新要求为准。
- 写完后自行检查：有没有不必要的高级词？有没有堆砌分点？是否符合作业要求？
